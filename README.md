# 🦠 COVID-19 Data Analysis

Exploratory data analysis project developed in Python using COVID-19 datasets.

---

## 🎯 Objective

Analyze the evolution of COVID-19 data across countries and time to identify trends and patterns.

---

## ⚙️ Analysis

The notebook includes:

- data cleaning and preprocessing  
- time-based analysis  
- country-level comparisons  
- calculation of key metrics  

---

## 💡 Analytical Focus

The project highlights how data can be used to understand real-world phenomena through:

- trend analysis  
- comparison across countries  
- interpretation of temporal patterns  

---

## 🛠 Tools

- Python  
- pandas  
- NumPy  

---

## 🇮🇹 Obiettivo

Analizzare l’andamento dei dati COVID nel tempo e tra diversi paesi, individuando trend e differenze significative.
